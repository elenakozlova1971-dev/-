# Семейный бюджет — статический сайт

Небольшой учёт доходов и расходов в браузере: приветственная страница и калькулятор с таблицей операций и итогами.

## Что внутри

- **`index.html`** — лендинг с кратким описанием и ссылками на калькулятор.
- **`budget.html`** — форма добавления записей, блок итогов, таблица с возможностью удалить строку или очистить всё.
- **`css/style.css`** — общие стили.
- **`js/budget.js`** — логика и сохранение в `localStorage`.

## Как открыть

1. Откройте **`index.html`** двойным щелчком в браузере, **или**
2. Из папки проекта запустите локальный сервер (например, `npx serve .`) и перейдите по адресу, который покажет терминал.

> При открытии файла напрямую (`file://`) всё должно работать так же; для некоторых сценариев удобнее HTTP-сервер.
